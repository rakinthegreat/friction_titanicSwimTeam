# WaitLess

Turn life's friction into focused productivity. This is a hybrid Next.js + Capacitor application designed to detect waiting periods and provide curated, time-boxed activities.

## Project Tracking & Collaboration

To keep our collaboration synced, please follow these guidelines when working on this project:

1.  **Check Progress**: Always read [`docs/PROGRESS.md`](docs/PROGRESS.md) to see what has been implemented so far.
2.  **Follow the Roadmap**: Refer to [`docs/TODO.md`](docs/TODO.md) for the current list of pending tasks.
3.  **Update as You Work**:
    *   Mark completed tasks in `docs/TODO.md`.
    *   Add a summary of your changes to `docs/PROGRESS.md` under the appropriate phase.
4.  **Agent Instructions**: If you are using an AI coding assistant, direct it to read `docs/PROGRESS.md` and `docs/TODO.md` as its first step to understand the current context.

## Tech Stack
- **Frontend**: Next.js 16 (React 19), TypeScript, Tailwind CSS
- **State**: Zustand + localforage (IndexedDB)
- **Mobile**: Capacitor JS
- **AI**: Google Gemini API
- **Native**: Java (Background services)
